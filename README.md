## 📖 About this project

The Sons-of-the-Forest-Build-Assistant provides tools to assist with construction and exploration in the game. This software removes obstacles so you focus on building. It serves as a manager for your game state. You modify health, stamina, hunger, thirst, and temperature. You gain access to all building materials and tools. Features include unlimited ammo, instant build, no collision, frozen logs, a full map, and teleportation.

## 🚀 Using the assistant

The application works while the game runs in the background. Follow this sequence for the best experience:

1. Launch your game and load your specific save file.
2. Minimize the game window by pressing the Windows key combined with the M key.
3. Open the BuildAssistant.exe file you saved earlier.
4. Select the check boxes for the features you want to activate.
5. Return to the game window to see your changes.

## ⚠️ Important considerations

Close the application before you perform a game update. Large game patches sometimes change how the game saves files. If the tools stop working, restart both the game and the assistant. Use a clean save file if you want to test these tools for the first time. Keep a backup copy of your game save files inside your game directory. This protects your progress if you make a mistake while modifying your settings.

## ❓ Frequently asked questions

Do I need to install anything else?
The application includes all necessary components to run on Windows 10 and 11. 

Is this tool safe to use?
The tool modifies only the local files that control your current game session. It does not access your personal accounts or private files on your computer.

Does the tool work with multiplayer?
The application functions primarily in single-player mode. Using these tools in multiplayer sessions might cause sync issues for other players.

How do I remove the tools?
Delete the .exe file to remove the program. Your game remains unchanged if you do not have the application active during your session.

Can I get banned for using this?
This tool is for offline play. Game servers usually detect modifications. Do not connect to public servers while using the tool to ensure your account remains in good standing.

What happens if the game crashes?
Close the assistant and restart the game. The files normally reset to their default state after a crash. 

Why does Windows show a warning?
Windows Defender sometimes flags programs that modify other running applications. This is a common precaution for helper software. You can click "More info" and then "Run anyway" if the system prevents the launch.

How do I update the tool?
Check the website periodically for new versions. When a new update appears, download the new file and replace the old version on your computer.
